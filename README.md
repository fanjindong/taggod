# 标签之神

`标签之神` 是一个 Chrome 标签页管理插件，面向经常同时打开几十到上百个标签页的用户。它用一个弹窗完成整理、搜索、去重、分组和会话保存，尽量减少在标签栏里来回找页面的时间。

## 功能特性

- 一键整理当前窗口标签页。
- 按域名创建 Chrome 原生标签分组。
- 自动尊重当前标签栏里的分组顺序。
- 星标重要分组，让它们整理后优先靠前。
- 关闭完整网址重复的标签页。
- 按标题、网址、域名搜索当前窗口标签页。
- 保存当前窗口会话。
- 恢复最近保存的会话。
- 使用快捷键整理标签页或保存会话。

## 整理规则

点击“整理标签页”后，插件会按下面的规则重新排列当前窗口：

1. 固定标签保持在最前面。
2. 星标分组排在非星标分组前面。
3. 多个星标分组之间，按当前标签栏从左到右的顺序排列。
4. 非星标分组之间，也按当前标签栏从左到右的顺序排列。
5. 无法识别顺序时，按域名排序兜底。
6. 同一个域名内保持原来的左右顺序。

这意味着：如果你已经在 Chrome 标签栏里手动把某个分组拖到前面，直接点击“整理标签页”即可，插件会尊重当前顺序。

## 优先分组

弹窗中的“优先分组”区域会展示当前窗口按域名聚合后的分组。

- 点击空心星标，可以把该分组设为优先分组。
- 点击实心星标，可以取消该分组的优先状态。
- 星标只决定分组是否进入优先区，不决定多个星标分组之间的顺序。

## 快捷键

默认快捷键：

- macOS：`Command+Shift+L` 打开插件弹窗。
- macOS：`Command+Shift+Y` 整理当前窗口标签页。
- macOS：`Command+Shift+S` 保存当前窗口会话。
- Windows 和 Linux：使用 `Ctrl` 替代 `Command`。

## 本地安装

1. 下载或克隆本项目。
2. 打开 Chrome 的 `chrome://extensions`。
3. 开启右上角的“开发者模式”。
4. 点击“加载已解压的扩展程序”。
5. 选择本项目目录。
6. 点击浏览器工具栏中的“标签之神”图标。

## 开发与校验

本项目目前不需要构建步骤，源码可以直接作为 Chrome 插件加载。

运行本地校验：

```bash
node scripts/校验插件.js
```

预期输出：

```text
插件文件校验通过
```

也可以分别检查脚本语法：

```bash
node --check background.js
node --check popup.js
```

## 权限说明

`标签之神` 只声明实现功能所需的 Chrome 权限：

- `tabs`：查询、激活、关闭、移动和创建标签页。
- `tabGroups`：创建和更新 Chrome 原生标签分组。
- `storage`：保存会话、分组快照和用户配置。

会话和配置保存在本地 `chrome.storage.local`，当前版本不包含云同步。
