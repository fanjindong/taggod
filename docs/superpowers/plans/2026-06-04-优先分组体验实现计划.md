# 优先分组体验实现计划

> **给智能代理工作者：** 逐项执行本计划，使用复选框语法跟踪每个步骤。

**目标：** 在弹窗中提供星标分组能力，并让一键整理自动识别当前标签栏顺序，把重要分组稳定排在前面。

**架构：** 后台服务继续负责排序、存储和批量标签操作，弹窗只负责展示当前分组和触发配置更新。优先分组配置保存在 `chrome.storage.local` 的 `settings.priorityGroups` 中，一键整理时读取星标状态，同时使用整理前当前窗口的分组顺序作为星标分组之间的排序依据。

**技术栈：** Manifest V3、原生 JavaScript、Chrome Extensions API、`chrome.tabs`、`chrome.storage.local`、Node.js 校验脚本。
