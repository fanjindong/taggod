# 分组显示名优化实现计划

> **给自动化执行者：** 本计划用于按任务逐步实现。每个步骤使用复选框跟踪，完成后必须运行对应校验。

**目标：** 让 Chrome 原生分组标题和弹窗展示使用更短的分组显示名，同时保持内部 `groupKey` 为完整主域名。

**架构：** `background.js` 继续作为分组规则唯一入口。新增显示名纯函数，整理、恢复、弹窗分组摘要和工作集快照复用同一映射逻辑，避免多个入口展示不一致。

**技术栈：** Manifest V3、Chrome `tabs` / `tabGroups` API、原生 JavaScript、本地 Node 校验脚本。
